 # ScratchGPT - Scratch AI Bridge

## Setup Instructions
1. Go to **Settings** > **Secrets and variables** > **Codespaces**.
2. Add these **Repository Secrets**:
   - `SCRATCH_USERNAME`: Your username.
   - `SCRATCH_SESSION_ID`: From your browser cookies.
   - `SCRATCH_PROJECT_ID`: The numbers in your Scratch URL.
   - `OPENROUTER_KEY`: Your key from openrouter.ai.

## Running the Bridge
1. Open this repo in a **GitHub Codespace**.
2. Wait for the terminal to finish installing requirements.
3. Type: `python main.py`
